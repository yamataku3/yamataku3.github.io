Project Alpha: A New HCI Paradigm

Hardware C++ Arduino

Overview

このプロジェクトは、触覚デバイスを用いた新しいコミュニケーションインターフェースの研究開発プロトタイプです。 ユーザーの直感的な動作をセンサーで捉え、遠隔地にいる相手に対して物理的なフィードバックとして伝達する仕組みを提案します。

Problem Statement

従来のビデオ通話システムでは視覚と聴覚の情報のみしかやり取りできないため、対面でのコミュニケーションに比べて没入感や親密さが欠如してしまうという課題がありました。

Approach & Implementation

私たちは、Arduinoを用いたカスタムのアクチュエーターモジュールを開発し、ユーザーの腕に装着できるウェアラブルデバイスを設計しました。 通信には低遅延のプロトコルを採用し、C++を用いてハードウェアの制御ソフトウェアを記述しています。

Results

プロトタイプの評価実験により、被験者の78%が既存のコミュニケーションツールよりも感情の伝達が豊かになったと報告しました。 この成果は国際会議において発表されました。
